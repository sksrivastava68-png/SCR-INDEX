Naming Convention — Everything in ONE Flat Folder
Source_Subject_YYMM.pdf
✓ RB_LCSafety_2604.pdf — source prefix auto-detected
✓ RB_HAGPromotion_2210.pdf — date auto-extracted
✓ DIV_NandedFireReport_2411.pdf — just type filename, fields auto-fill
✓ UNION_WageRevision_2603.pdf — tap department chips to tag
✓ STATE_LandAcquisition_2511.pdf — findable from any angle
All files live in: OneDrive / SCR Work / All Documents / YYYY /
One subfolder per year only — no subject or source folders needed.
